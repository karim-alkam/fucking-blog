11:T3171,<p>You’re running a long update on a remote server, your internet blips, and the connection drops. The process is dead, and you have no idea if it finished. Or maybe you have five different terminal windows open, trying to juggle monitoring logs, editing files, and running commands.</p>
<p>Enter <strong>tmux</strong>.</p>
<p>Tmux (Terminal Multiplexer) allows you to have terminals inside of terminals. It keeps your sessions persistent (so they survive connection drops) and lets you split your screen into multiple panes and windows.</p>
<p>In this guide, I’m going to walk you through the basics, and then I’m going to share a custom configuration that makes tmux 10x easier to use.</p>
<h2 id="part-1-getting-started--the-defaults">Part 1: Getting Started &amp; The Defaults</h2>
<p>First, let&#39;s make sure it&#39;s installed. On Debian/Ubuntu, it’s as simple as:</p>
<pre><code class="language-Bash">sudo apt install tmux
</code></pre>
<p>To start tmux, just type <code>tmux</code> in your terminal. You’ll know you’re in tmux because of the distinct green status bar at the bottom.</p>
<p><img src="/images/Pasted%20image%2020251227015049.png" alt=""></p>
<h3 id="the-prefix-key">The &quot;Prefix&quot; Key</h3>
<p>To talk to tmux (instead of the shell), you have to send a <strong>Prefix Key</strong>. By default, this is <strong><code>Ctrl + b</code></strong>. You hold Control, press <code>b</code>, release both, and then press your command key.</p>
<h3 id="panes-splitting-the-screen">Panes (Splitting the Screen)</h3>
<p>The real magic happens when you split your window.</p>
<ul>
<li><strong>Vertical Split:</strong> Prefix + <code>%</code></li>
<li><strong>Horizontal Split:</strong> Prefix + <code>&quot;</code></li>
</ul>
<p>You can navigate between them using <code>Prefix</code> + <code>Arrow Keys</code>.</p>
<p><img src="/images/Pasted%20image%2020251227015026.png" alt=""></p>
<h3 id="windows-tabs">Windows (Tabs)</h3>
<p>If your screen gets too cluttered, you can create new Windows (like tabs).</p>
<ul>
<li><strong>Create Window:</strong> Prefix + <code>c</code></li>
<li><strong>Next/Prev Window:</strong> Prefix + <code>n</code> or <code>p</code></li>
<li><strong>Rename Window:</strong> Prefix + <code>,</code></li>
</ul>
<p><img src="/images/Pasted%20image%2020251227015303.png" alt=""></p>
<h3 id="sessions-the-lifesaver">Sessions (The Lifesaver)</h3>
<p>The most powerful feature of tmux is session management. If you detach from tmux (Prefix + <code>d</code>), your processes keep running in the background. You can log out, go to lunch, come back, and pick up exactly where you left off.</p>
<p>Here are the essential commands to manage your terminal life:</p>
<ol>
<li>Creating a Named Session</li>
</ol>
<p>While typing tmux works, it gives the session a boring number (0, 1, 2...). It is much better to give it a name so you remember what project is running inside:</p>
<pre><code class="language-Bash">tmux new -s my-new-session
</code></pre>
<ul>
<li><strong>Note:</strong> or you can just use <code>tmux</code> alone and it will create an unnamed seesion</li>
</ul>
<ol start="2">
<li>Listing Sessions</li>
</ol>
<p>To see what sessions are currently running in the background, use:</p>
<pre><code class="language-Bash">tmux ls
# OR the long version
tmux list-sessions
</code></pre>
<ul>
<li><strong>Tip</strong>: If you run <code>tmux ls</code> while you are already <strong>inside</strong> a session, it will list all sessions and explicitly mark the one you are currently attached to.</li>
</ul>
<ol start="3">
<li>Attaching to Sessions</li>
</ol>
<p>To get back into a session, you have a few options:</p>
<pre><code class="language-Bash"># Attaches to the most recently used session
tmux a 
# OR
tmux attach

# Attaches to a SPECIFIC named session (e.g., &quot;my-session&quot;)
tmux a -t my-session
</code></pre>
<ol start="4">
<li>Killing the Server</li>
</ol>
<p>If you want to close all active sessions and stop tmux entirely, run:</p>
<pre><code class="language-Bash">tmux kill-server
</code></pre>

<h2 id="part-2-supercharging-tmux-the-configuration">Part 2: Supercharging Tmux (The Configuration)</h2>
<p>The defaults are functional, but they require some &quot;finger gymnastics.&quot; Reaching for <code>%</code> or <code>&quot;</code> to split screens isn&#39;t intuitive, and <code>Ctrl + b</code> is a bit of a stretch.</p>
<p>We are going to create a configuration file to remap our keys, add mouse support, and make the interface look much better.</p>
<p>Create a file in your home directory called <code>.tmux.conf</code>:</p>
<pre><code class="language-Bash">nano ~/.tmux.conf
</code></pre>
<p>Paste the following configuration into that file. This setup changes the prefix to <code>Ctrl + j</code>, enables mouse mode, and adds standard <code>Alt</code> key navigation.</p>
<h3 id="my-tmuxconf">My <code>.tmux.conf</code></h3>
<pre><code class="language-Bash">set -sg escape-time 50

# Initial setup
set -g default-terminal xterm-256color
set -g status-keys vi

# --- KEY REMAPPING ---
# Change Prefix to C-j and C-f (Easier to reach)
set-option -g prefix C-j
unbind-key C-j
bind-key C-j send-prefix
set -g base-index 1

# Use Alt-arrow keys WITHOUT PREFIX to switch panes
bind -n M-Left select-pane -L
bind -n M-Right select-pane -R
bind -n M-Up select-pane -U
bind -n M-Down select-pane -D

# Set easier window split keys (v for vertical, h for horizontal)
bind-key v split-window -h
bind-key h split-window -v

# Shift arrow to switch windows
bind -n S-Left  previous-window
bind -n S-Right next-window

# Easily reorder windows with CTRL+SHIFT+Arrow
bind-key -n C-S-Left swap-window -t -1
bind-key -n C-S-Right swap-window -t +1

# Synchronize panes (Type in all panes at once)
bind-key y set-window-option synchronize-panes\; display-message &quot;synchronize mode toggled.&quot;

# Easy config reload
bind-key r source-file ~/.tmux.conf \; display-message &quot;tmux.conf reloaded.&quot;

# Easy clear history
bind-key L clear-history

# --- COPY MODE &amp; MOUSE ---
setw -g mode-keys vi
unbind p
bind p paste-buffer
bind-key -T copy-mode-vi &#39;v&#39; send -X begin-selection
bind-key -T copy-mode-vi &#39;y&#39; send -X copy-selection-and-cancel

# Enable Mouse Mode (Click to select panes, resize, scroll)
set -g mouse on



# -- The Theme --

set -g default-terminal &quot;tmux-256color&quot;

# 1. Configure the theme (MUST BE BEFORE LOADING THE PLUGIN)
set -g @catppuccin_flavor &quot;mocha&quot;
set -g @catppuccin_window_status_style &quot;rounded&quot;

# --- Fix for Window Names ---
set -g @catppuccin_window_text &quot; #W&quot;
set -g @catppuccin_window_default_text &quot; #W&quot;
set -g @catppuccin_window_current_text &quot; #W&quot;

# Change the background of the text part to Mauve
set -g @catppuccin_window_current_background &quot;#{@thm_mauve}&quot;

# Change the background of the number part to Mauve (matches the text part)
set -g @catppuccin_window_current_number_color &quot;#{@thm_mauve}&quot;

# Ensure the text on the Mauve background is dark (readable)
set -g @catppuccin_window_current_text_color &quot;#{@thm_bg}&quot;


# 2. Load the plugin
run ~/.config/tmux/plugins/catppuccin/tmux/catppuccin.tmux

# 3. Configure the Status Bar (Right Side)
set -g status-right-length 100
set -g status-left-length 100
set -g status-left &quot;&quot;
set -g status-right &quot;&quot;
set -ag status-right &quot;#{E:@catppuccin_status_session}&quot;
set -ag status-right &quot;#{E:@catppuccin_status_date_time}&quot;
</code></pre>
<h3 id="applying-the-changes">Applying the Changes</h3>
<p>Once you save the file, you need to tell tmux to look at it. If you are already in tmux, you can&#39;t use the new reload key yet because the config isn&#39;t loaded!</p>
<p>The easiest way is to kill your current tmux server: <code>tmux kill-server</code> and start fresh. Or, from inside tmux, run:</p>
<pre><code class="language-Bash">tmux source-file ~/.tmux.conf
</code></pre>

<ul>
<li><strong>Note</strong>: these are not the colors specified in the conf file, this is because my Windows terminal applies it&#39;s own colors</li>
</ul>

<h2 id="part-3-the-new-workflow">Part 3: The New Workflow</h2>
<p>With this configuration file, your life just got a lot easier. Here is how your workflow changes:</p>
<ol>
<li><strong>Prefix:</strong> You can now use <strong><code>Ctrl + j</code></strong>.</li>
<li><strong>Splitting:</strong> Use Prefix + <strong><code>v</code></strong> for Vertical and Prefix + <strong><code>h</code></strong> for Horizontal. Logical!</li>
<li><strong>Navigation:</strong> No prefix needed! Just hold <strong><code>Alt</code> + Arrow Keys</strong> to jump between panes.</li>
<li><strong>Mouse Mode:</strong> You can now click on panes to select them and drag borders to resize them.</li>
</ol>
