      <p>You’re running a long update on a remote server, your internet blips, and the connection drops. The process is dead, and you have no idea if it finished. Or maybe you have five different terminal windows open, trying to juggle monitoring logs, editing files, and running commands.</p>
<p>Enter <strong>tmux</strong>.</p>
<p>Tmux (Terminal Multiplexer) allows you to have terminals inside of terminals. It keeps your sessions persistent (so they survive connection drops) and lets you split your screen into multiple panes and windows.</p>
<p>In this guide, I’m going to walk you through the basics, and then I’m going to share a custom configuration that makes tmux 10x easier to use.</p>
<h2 id="part-1-getting-started--the-defaults">Part 1: Getting Started &amp; The Defaults</h2>
<p>First, let's make sure it's installed. On Debian/Ubuntu, it’s as simple as:</p>

  <p>To start tmux, just type <code>tmux</code> in your terminal. You’ll know you’re in tmux because of the distinct green status bar at the bottom.</p>
<p><img src="/images/Pasted%20image%2020251227015049.png" alt=""></p>
<h3 id="the-prefix-key">The "Prefix" Key</h3>
<p>To talk to tmux (instead of the shell), you have to send a <strong>Prefix Key</strong>. By default, this is <strong><code>Ctrl + b</code></strong>. You hold Control, press <code>b</code>, release both, and then press your command key.</p>
<h3 id="panes-splitting-the-screen">Panes (Splitting the Screen)</h3>
<p>The real magic happens when you split your window.</p>
<ul>
<li><strong>Vertical Split:</strong> Prefix + <code>%</code></li>
<li><strong>Horizontal Split:</strong> Prefix + <code>"</code></li>
</ul>
<p>You can navigate between them using <code>Prefix</code> + <code>Arrow Keys</code>.</p>
<p><img src="/images/Pasted%20image%2020251227015026.png" alt=""></p>
<h3 id="windows-tabs">Windows (Tabs)</h3>
<p>If your screen gets too cluttered, you can create new Windows (like tabs).</p>
<ul>
<li><strong>Create Window:</strong> Prefix + <code>c</code></li>
<li><strong>Next/Prev Window:</strong> Prefix + <code>n</code> or <code>p</code></li>
<li><strong>Rename Window:</strong> Prefix + <code>,</code></li>
</ul>
<p><img src="/images/Pasted%20image%2020251227015303.png" alt=""></p>
<h3 id="sessions-the-lifesaver">Sessions (The Lifesaver)</h3>
<p>The most powerful feature of tmux is session management. If you detach from tmux (Prefix + <code>d</code>), your processes keep running in the background. You can log out, go to lunch, come back, and pick up exactly where you left off.</p>

<h2 id="part-2-supercharging-tmux-the-configuration">Part 2: Supercharging Tmux (The Configuration)</h2>
<p>The defaults are functional, but they require some "finger gymnastics." Reaching for <code>%</code> or <code>"</code> to split screens isn't intuitive, and <code>Ctrl + b</code> is a bit of a stretch.</p>
<p>We are going to create a configuration file to remap our keys, add mouse support, and make the interface look much better.</p>
<p>Create a file in your home directory called <code>.tmux.conf</code>:</p>

    <div class="relative group">
      <pre class="custom-code-scrollbar !p-4"><code class="hljs language-Bash !p-1 !m-1 !mx-2">nano ~/.tmux.conf</code></pre>
      <button class="copy-btn absolute top-2 right-2 p-1 rounded-sm bg-gray-700 hover:bg-gray-600 transition-colors opacity-0 group-hover:opacity-100 focus:opacity-100" data-code="nano ~/.tmux.conf" aria-label="Copy code" title="Copy code">
        <svg xmlns="http://www.w3.org/2000/svg" class="h-4 w-4 text-gray-300 pointer-events-none" viewBox="0 0 20 20" fill="currentColor">
          <path d="M8 3a1 1 0 011-1h2a1 1 0 110 2H9a1 1 0 01-1-1z"></path>
          <path d="M6 3a2 2 0 00-2 2v11a2 2 0 002 2h8a2 2 0 002-2V5a2 2 0 00-2-2 3 3 0 01-3 3H9a3 3 0 01-3-3z"></path>
        </svg>
      </button>
    </div>
  <p>Paste the following configuration into that file. This setup changes the prefix to <code>Ctrl + j</code>, enables mouse mode, and adds standard <code>Alt</code> key navigation.</p>
<h3 id="my-tmuxconf">My <code>.tmux.conf</code></h3>

    <div class="relative group">
      <pre class="custom-code-scrollbar !p-4"><code class="hljs language-Bash !p-1 !m-1 !mx-2"><span class="hljs-built_in">set</span> -sg escape-time 50

<span class="hljs-comment"># Initial setup</span>
<span class="hljs-built_in">set</span> -g default-terminal xterm-256color
<span class="hljs-built_in">set</span> -g status-keys vi

<span class="hljs-comment"># --- KEY REMAPPING ---</span>
<span class="hljs-comment"># Change Prefix to C-j and C-f (Easier to reach)</span>
set-option -g prefix C-j
unbind-key C-j
bind-key C-j send-prefix
<span class="hljs-built_in">set</span> -g base-index 1

<span class="hljs-comment"># Use Alt-arrow keys WITHOUT PREFIX to switch panes</span>
<span class="hljs-built_in">bind</span> -n M-Left select-pane -L
<span class="hljs-built_in">bind</span> -n M-Right select-pane -R
<span class="hljs-built_in">bind</span> -n M-Up select-pane -U
<span class="hljs-built_in">bind</span> -n M-Down select-pane -D

<span class="hljs-comment"># Set easier window split keys (v for vertical, h for horizontal)</span>
bind-key v split-window -h
bind-key h split-window -v

<span class="hljs-comment"># Shift arrow to switch windows</span>
<span class="hljs-built_in">bind</span> -n S-Left  previous-window
<span class="hljs-built_in">bind</span> -n S-Right next-window

<span class="hljs-comment"># Easily reorder windows with CTRL+SHIFT+Arrow</span>
bind-key -n C-S-Left swap-window -t -1

<h2 id="part-3-the-new-workflow">Part 3: The New Workflow</h2>
<p>With this configuration file, your life just got a lot easier. Here is how your workflow changes:</p>
<ol>
<li><strong>Prefix:</strong> You can now use <strong><code>Ctrl + j</code></strong>.</li>
<li><strong>Splitting:</strong> Use Prefix + <strong><code>v</code></strong> for Vertical and Prefix + <strong><code>h</code></strong> for Horizontal. Logical!</li>
<li><strong>Navigation:</strong> No prefix needed! Just hold <strong><code>Alt</code> + Arrow Keys</strong> to jump between panes.</li>
<li><strong>Mouse Mode:</strong> You can now click on panes to select them and drag borders to resize them.</li>
</ol>
